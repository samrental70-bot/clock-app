# Store Submission Testing Checklist

## Accounts and Access
- Create demo owner account.
- Create demo employee account.
- Create demo company.
- Confirm login works.
- Confirm demo credentials are safe for reviewers.

## Core App Flows
- Test clock in/out.
- Test location prompt.
- Test schedule accept/decline.
- Test timesheets.
- Test photo upload.
- Test receipt upload.
- Test Team/Menu.
- Test notifications if enabled.
- Test role-based access for owner/supervisor/employee.

## PWA and Browser Testing
- Test PWA install prompt.
- Test mobile Safari.
- Test Android Chrome.
- Test desktop browser load.

## Native Wrapper Testing
- Test iOS wrapper if created.
- Test Android wrapper if created.
- Verify camera/photo permissions.
- Verify location permissions.
- Verify notifications.
- Verify file upload behavior.
- Verify app resumes from background.

## Store Readiness
- Verify privacy policy URL is live.
- Verify support URL is live.
- Verify screenshots show real app UI.
- Verify screenshots use safe demo data.
- Verify no secrets in repo.
- Verify production backend config is safe.
- Verify App Store / Play Store declarations match actual behavior.
