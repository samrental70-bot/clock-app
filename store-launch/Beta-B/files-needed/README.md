# Files Needed for Store Launch

Place manually prepared store assets here only if they are safe to commit.

Do not place secrets in this folder.

Do not commit:
- `.env` files.
- API keys.
- Private keys.
- Android keystores.
- iOS certificates.
- Provisioning profiles.
- Any private customer data.

Expected safe assets may include:
- Final public app icon source.
- Screenshot exports using safe demo data.
- Feature graphic source/export.
- Public metadata copy drafts.
